## 使用提醒
请先安装 python3 和 pip 工具，并配置正确的环境变量。
[🔨]请先安装必需的第三方库：aiohttp。
键入`python3 main.py`开始运行。

使用脚本时，请保证网络连接通畅。为保持稳定，建议连接校园内网。
